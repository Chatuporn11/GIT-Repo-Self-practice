1. (...a) → rest parameter เก็บค่าที่ส่งเข้ามาเป็น array เช่น Sum(1,2,3) → a = [1,2,3]
arguments → object ที่มีค่าพารามิเตอร์ทั้งหมดที่ส่งเข้ามา (คล้าย array)
ลูปเริ่มจาก i = 1 ไม่ใช่ 0 → เลย ข้าม argument ตัวแรกเสมอ
ผลรวมจะไม่รวมค่าตัวแรก!


2. FunctionMain รับ parameter เป็น ฟังก์ชัน (func)
return func(2) → เรียกฟังก์ชันนั้นด้วยค่า 2
ที่ส่งเข้าไปคือ (e)=>e*e → ฟังก์ชันที่คูณตัวเอง
